## Condiciones 

- f continua, f = 0
- [a,b] tq se cumple **Bolzano** -> ∃ c ∈ (a,b) / f(c) = 0
- f(a) y f(b) tienen != signo 

## Lo que entra por teclado

f       --| datos  
[a, b]    | de   
tal     --| entrada 

> Ejemplo

```python
a=18.063
b=18.147

M = (a + b) / 2 
R = (a - b) / 2

#  Salida "en bruto"
print(M)    # Cota inferior   
print(R)    # Cota superior

#  Errores 

M_prima = round(M, 2)
R_prima = round(R, 2)

print(M_prima)
print(R_prima)
```
